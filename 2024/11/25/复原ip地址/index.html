复原ip地址

思路

  • 经典回溯算法，可以拆分成回溯三部曲
  • 1、首先，明确递归函数的参数，这里我们需要传入string& s, int startIndex, int pointNum
  • 2、其次，明确递归终止条件，当我们的pointNum = 3，并且剩下的字符串合法时，终止；
  • 3、另外，每层递归中的逻辑应该是先找到断点，然后后面继续递归；
  • 4、合法函数需要满足一些条件，例如范围，首0等；

时间复杂度

  • 时间复杂度是O(3^SEG_COUNT ×∣s∣)，ip地址每一段位数不超过3，因此递归每一层，我们最多只会深入到下一层的3种情况，递归层数最大为4层，我们将ip地址放入答案数组中需要o(|S|)。

空间复杂度

  • 空间复杂度是o(SEG_COUNT)，因为我们只额外使用了长度为SEG_COUNT的数组存储已经搜索过的ip地址。
